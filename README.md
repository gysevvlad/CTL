C Template Library
==================

Библиотека шаблонов СИ.

Состав
======

На данный момент библиотека включает следующие стабильные модули:
  - [iterator][]

Требуют доработки:
  - pair
  - linked_list
  - map

[iterator]: https://github.com/gysevvlad/CTL/blob/master/ITERATOR.md

В планах:
  - node
  - tree node
  - vector
  - binary search tree
  - treap (декартово дерево)
  - lazy list
  - lazy vector
